pHKondo fluxo de trabalho

Adicionar condomínio (s) -> Criar Condomínio
Selecione detalhes do condomínio

Vá para Frações
Adicione propriedades -> Criar Fração
Digite Detalhes sobre a propriedade

Ir para Proprietários
Adicionar proprietários -> Criar Proprietário
Ou use Adicionar Proprietário para pesquisar proprietários já registados na sua base de dados

Regresse aos detalhes do condomínio
Ir para Administradores
Adicionar administradores -> Adicionar Administrador
* Administrador é o proprietário (s) selecionado para ter uma relação direta entre a gestão do Condomínio/empresa/serviço e todos os proprietários em vários assuntos, como conta bancária controlo, manutenção, queixas.

Regresse aos detalhes do condomínio
Ir para Exercícios
Adicionar exercício -> Criar exercício
* Exercício/Ano Fiscal é usado para configurar o seu período de gestão, que pode o período que entender, não necessariamente um ano.
É usado para controlar:
- Saldos de contas bancárias, Orçamentos...
- Relatórios para Balancete, receitas, despesas, em dívida, mapa de quotas

Regresse aos detalhes do condomínio
Ir para contas
Adicionar contas bancárias -> Nova Conta
Digite Detalhes da conta

Ir para Movimentos
Adicionar movimentos de conta -> Criar Movimento
* Cada movimento terá de pertencer ao exercício ativo
* Para cada exercício você deve criar o primeiro e último movimento descrito abaixo, isso permitirá que o pHKondo possa realizar os relatórios baseados em exercício.
* O primeiro movimento deve ser:
Montante inicial para balanço de abertura no ano fiscal
Abertura / Encerramento na Categoria
Saldo na Operação Movimento
Tipo de movimento: Crédito se balança aberto é 0 ou superior, se Debito saldo em aberto é <0
* O último movimento deve ser:
Abertura / Fecho de Categoria
Abertura Balancete na Operação

Regresse aos detalhes do condomínio
Ir para Orçamentos
Criar orçamentos -> Criar Orçamento
Defina os seus orçamentos, use os campos da Valor, Fundo Comum de Reserva%), Data de Início, Quotas, Periodicidade, Distribuição, Prazo de pagamento para depois criar notas de débito de forma automática automática, com base nas características do orçamento.
Para notas de débito automático (Criar Notes) orçamento deve estar no estado ativo.

Ir para Manutenções para gerir as programações de manutenção ao condomínio
Ir para Seguros de gerir seguros de condomínio e frações
Ir para Anexos para gerir os documentos anexados
Ir para os relatórios para Balancete, Receitas, Despesas, Conta Corrente, Mapas...
Ir para Minutas para imprimir alguns documentos usados na gestão do condomínio...
